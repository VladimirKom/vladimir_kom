Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал зхватывать сторую статью, как показано в данном примере

Статья 1

Размер блока: Занимает всю достуаную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom

Статья 2

Размер блока: Занимает всю достуаную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширишу. Высота определяется помещенным внутрь контентом
